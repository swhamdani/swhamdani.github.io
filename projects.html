Blockchain-Enhanced Network Scanning & Monitoring (BENSAM) Tool
Sep 2025 – Present
Lead Researcher & Developer Python Blockchain Network Security Published

A next-generation, enterprise-grade network scanning and monitoring framework that integrates blockchain-based logging for tamper-proof auditability. The system performs comprehensive network discovery, device profiling, and vulnerability assessment while integrating external threat intelligence sources for enriched analysis.

  • Key innovation: blockchain technology ensures integrity, transparency, and non-repudiation of scan results and security events — addressing a critical gap in conventional monitoring tools where logs can be altered post-incident.
  • Modular, extensible architecture supporting real-time monitoring, automated reporting, and compliance-oriented secure storage for forensic and regulatory purposes.
  • Smart contract logic enables automated anomaly flagging without reliance on a centralized trust authority.
  • Framework described and validated in the published journal paper: Blockchains, 4(1), 1. (2025). Manuscript extending to policy-aware vulnerability assessment currently in preparation.

AI-Based Web Defacement Detection Bot (AI-WDDB)
Oct 2019 – Mar 2020
Team Lead Python Machine Learning Computer Vision NCSAEL

Led the design and development of an ML-driven automated bot for continuous detection of website defacement attacks — a prevalent form of hacktivism targeting government and critical-sector websites in Pakistan.

  • Monitored visual and structural page changes using image comparison and DOM analysis techniques to detect unauthorized modifications.
  • Evaluated multiple ML classification algorithms; selected the optimal model based on precision, recall, and false-positive rate on a corpus of Pakistani government and critical-sector websites.
  • Enabled near-real-time alerting, replacing time-consuming manual monitoring workflows.
  • Achieved high-precision defacement detection with a minimal false-positive rate.

MS Thesis Research

DTMSim-IoT: Distributed Trust Management Simulator for IoT Networks
Feb 2018 – Dec 2018
Principal Investigator — MS Thesis C# .NET IoT Trust Management Open Source Grade A

Conceived, designed, and implemented a standalone .NET-based simulation framework to model and evaluate dynamic trust propagation across heterogeneous IoT topologies under adversarial conditions — the central contribution of an MS thesis awarded Grade A at NUST.

  • Problem addressed: IoT deployments are vulnerable to nodes that behave well initially to accumulate positive trust, then exploit that trust to attack peer nodes (On-Off attacks). No standardized simulation tool existed to test trust management schemes against this behavior at scale.
  • DTMSim-IoT: enables researchers to implement any static or dynamic trust management model, simulate multi-service IoT network topologies, and evaluate trust convergence and attack resilience.
  • DDTMS scheme: proposed Dynamic Distributed Trust Management Scheme successfully detects On-Off attacks and classifies misbehaving nodes into low, mild, and severe severity levels for targeted isolation.
  • Simulation results show improved performance over state-of-the-art schemes in time-to-detection and interaction efficiency.
  • Published at IEEE DASC 2020 (8 citations) and Turkish Journal of Electrical Engineering (11 citations).
