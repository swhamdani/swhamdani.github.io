Industry and Applied R&D

Blockchain-Enhanced Network Scanning and Monitoring (BENSAM) Tool

Sep 2025 to April 2026
Lead Researcher and Developer Python Blockchain Network Security

A network scanning and monitoring framework that integrates blockchain-based logging for tamper-proof auditability. The system performs network discovery, device profiling, and vulnerability assessment while integrating external threat intelligence sources. Smart contract logic enables automated anomaly flagging without a centralized trust authority. The framework is designed with a modular architecture supporting real-time monitoring, automated reporting, and secure storage of scan logs for forensic and compliance purposes.

Published in Blockchains, 4(1), 1 (2025). A conference paper extending this work to policy-aware vulnerability assessment is in preparation.

MedScanner: Med7 Clinical NLP Model Integration

Apr 2023 to Aug 2023
Individual Contributor Python Flask spaCy Clinical NLP

Developed a RESTful API consumed by a React Native mobile application (iOS and Android) that uses clinical NLP to extract structured medication information from unstructured clinical text. Integrated Med7, a clinical named-entity recognition model, to extract seven medication attributes: drug name, dosage, route, frequency, duration, form, and strength. Designed for electronic health record workflows to reduce manual data entry burden.

Academic Thesis Research

DTMSim-IoT: Distributed Trust Management Simulator for IoT Networks

Feb 2018 to Dec 2018
Principal Investigator, MS Thesis C# .NET IoT Open Source Thesis Grade: A

The central problem addressed is that IoT nodes can behave correctly initially to accumulate positive trust, then exploit that trust to mount On-Off attacks against peers. No standardized simulation tool existed to test trust management schemes against this behavior at scale. This work produced two contributions:

  • DTMSim-IoT: a .NET-based simulation framework enabling researchers to implement any static or dynamic trust management model, simulate heterogeneous IoT topologies, and evaluate trust convergence and attack resilience. Open-sourced on GitHub.
  • DDTMS scheme: the proposed Dynamic Distributed Trust Management Scheme detects On-Off attacks and classifies misbehaving nodes into low, mild, and severe severity levels. Simulation results show improved time-to-detection over comparable state-of-the-art schemes.

Published at IEEE DASC 2020 (8 citations) and in the Turkish Journal of Electrical Engineering and Computer Sciences (11 citations).
